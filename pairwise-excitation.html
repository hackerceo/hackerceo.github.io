A pairwise excitation system uses an multi-conductor cable to selectively activate effects by treating each conductor as a node in a matrix, where an effect is triggered by applying +5 V to one conductor and ground to another, driving the LED inside an optoisolator. A controller chooses exactly one “source” conductor (driven high) and one “sink” conductor (pulled low), while ensuring all other conductors remain in a high-impedance (floating) state so no unintended current paths exist. On the software side, each possible conductor pair is encoded as a single “activation address,” which is mathematically mapped to a unique (positive / negative) conductor pair, allowing simple and scalable control of many effects using only a limited number of wires. 
            For 𝑁 = conductorCount:

            totalAddresses = 𝑁 * (𝑁−1)

            Each address corresponds to one valid (𝑖,𝑗) where 𝑖 ≠ 𝑗.
        
            Let:
                a = activationAddress
                N = conductorCount

            Compute positive conductor:
                positive = (a / N−1)

            Compute intermediate index:
                k = a mod (N−1)

            Compute negative conductor:
                if k < positive then negative = k
                if k >= positive then negative = k + 1
